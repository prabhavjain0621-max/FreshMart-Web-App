# 🛒 Fresh Mart - Mall

An interactive grocery mart web application built using **HTML5, CSS3, and Vanilla JavaScript**. The project provides a game-like shopping experience where users can walk through a virtual supermarket, collect grocery items, manage a shopping cart, and complete checkout.

---

## 📌 Project Overview

Fresh Mart - Mall is a front-end web project that simulates a real supermarket. Users navigate the store, interact with different product shelves, add items to their trolley, view a live bill, and complete their shopping through an engaging interface.

---

## ✨ Features

* 🛍️ Interactive grocery shopping experience
* 🚶 Player movement using keyboard or on-screen controls
* 🥦 Multiple product categories

  * Fruits & Vegetables
  * Snacks & Beverages
  * Dairy Products
* 🛒 Add products to shopping trolley
* 🧾 Dynamic shopping bill generation
* 💰 Automatic GST calculation
* 💳 Checkout simulation
* 🎉 Order confirmation popup
* 📱 Responsive design for desktop and mobile
* 🎨 Modern UI with animations and smooth transitions

## 🛍️ Shopping Flow

1. Walk through different grocery aisles.
2. Move close to a product.
3. Press **Grab** to view product details.
4. Add the item to your trolley.
5. Open the shopping bill.
6. Review subtotal, GST, and total amount.
7. Complete checkout.
8. Receive the order confirmation message.
